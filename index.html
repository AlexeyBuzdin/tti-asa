    TTI Android Schedule Application

    Android schedule application for Transport and Telecommunication Institute

    Goal of the project is to create a simple application for TTI student to view their lesson schedule. Project in is early stage of development and will be completed in few stages.

    Release notes:

    • Version 0.1 (private beta) - m.tsi schedule ported to Android as it is now, with cached student code
    • Version 0.2 - offline schedule cacheing
    • Version 1.0 - compact calendar to ease the browsing proccess
    • Version 1.1 - ?integration with google calendar?
    Anyone can share his thoughts about the project and their ideas could be added to the project. All release note can be subject to changes.
